To ensure code quality and consistent styling:

### Formatting
Automatically format all source files using Prettier:

// turbo
1. Run `pnpm run format`.

### Linting
Check for code quality issues and potential bugs using ESLint:

// turbo
1. Run `pnpm run lint`.

### Type Checking
Manually run the TypeScript compiler to check for type errors:

// turbo
1. Run `tsc --noEmit`.
